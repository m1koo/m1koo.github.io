Java算法之--股神

作者:Miko    发布于: 2017-05-02

博主前一段时间参加了百度的在线考试，哭的稀里糊涂的，发觉自己算法这一方面欠缺的真的不是一星半点。接下来一段时间打算每天（三天打鱼两天晒网=-=）都做几道算法题。是时候提高一下自己了。
给大家一个在线测试的网址：赛码网
OK，废话不多说上题目。

题目很简单，简单来说是一个小赛同学经过一段时间的算法学习，走向人生巅峰，迎娶白富美的扯淡的故事。
这种题目咋一看很简单，但一想有很麻烦。博主一上来就开始惯性思维，for循环，一个？必然不够，两个，一个记录总数一个递增。但是很不幸运行超时，代码也不够优雅。这时候就需要画图大法了。

宝宝一眼就看出（冥思苦想=-=），可以将数列分为几组，只要我们计算出下降的天数，那么最终使用总天数-2*下降天数即可。OK思路定下来了，那么怎么计算下降的天数呢？
两种方法：

  1. for循环

  2. 开方解一元二次方程

    下面就是撸代码了：

1.for循环方法

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        while(in.hasNext()){
            Integer n = in.nextInt();
            for(int i = 1;;i++){
                int sum = 2*i+i*(i-1)/2;
                if(sum>n||sum==n){
                    System.out.println(n-(i-1)*2);
                    break;
                }
            }
        }
    }
}

2.开方

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
        int n = sc.nextInt();
        int day = (int) (-1.5+Math.sqrt(9+8*(n-1))/2);
        System.out.println(n-day*2);
    }
    }
}

两种方法其实就是正反两种思路解一元二次方程。
